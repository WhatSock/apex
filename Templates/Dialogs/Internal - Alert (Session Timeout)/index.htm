Session Timeout Alert Dialog Internal Content

IMPORTANT: YOUR SESSION IS ABOUT TO EXPIRE.

|

The dialog is optimized for screen reader and keyboard only users:

  • Activate the Login button to open the dialog.
  • Screen reader users will be confined within the dialog content, and will not be able to see the background page content. (Confirmed in both Windows platform and iOS touch screen devices)
  • Keyboard focus is automatically restricted to active elements within the dialog.
