Standard Switches

  • Thermostat
  • Security
  • Lights

The simulated switches are optimized for screen reader and keyboard only users:

  • Set focus to the switches using the keyboard.
  • Press Enter or Space to activate a switch.
  • Mouse users can click the desired switch to activate as usual.
