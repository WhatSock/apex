Reading List

    The tree is optimized for screen reader and keyboard only users:

    • Activate the triggering element using the keyboard.
    • Press the Up and Down Arrow keys to move focus between the tree options.
    • Press the Left and Right Arrow keys to open or close subtrees.
    • Press any letter or number to jump to a tree item starting with that character or digit.
    • Press Enter to activate a tree option.
    • Press Escape to close the currently open tree.
    • Press Tab or set focus back on the triggering element to close the tree and all subtrees.
    • Mouse users can click the desired tree option as usual.
