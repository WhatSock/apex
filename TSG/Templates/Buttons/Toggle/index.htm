Toggle Buttons

  • Like

    Nobody expects the Spanish Inquisition.

The simulated toggle buttons are optimized for screen reader and keyboard only users:

  • Set focus to the buttons using the keyboard.
  • Press Enter or Space to toggle a button.
  • Mouse users can click the desired button to activate as usual.
